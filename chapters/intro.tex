本组希望设计的音乐播放系统[下称MyPlayer]是一款windows平台的应用程序，它至少应该支持以下功能：
\begin{itemize}
 \item   在线音乐播放
 \item   歌单管理
 \item   评价
 \item   个性化推荐
 \item   分享
 \item   音乐购买/下载/上传
 \item   歌曲排行榜
\end{itemize}
通过音乐社交网络，在流式无线电台、用户的计算机或一些便携式音乐设备上记录用户收听的所有歌曲的细节，建立每个用户的音乐品味的档案。
